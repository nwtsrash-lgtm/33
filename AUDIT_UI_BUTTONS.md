## 3) دورة حياة المنتج (الموثّقة فعلاً)

```
منافس جديد → كشط → تحليل → تصنيف إلى قسم (أعلى/أقل/موافق/مفقود/مراجعة/مستبعد)
   → [تعديل القيمة في البطاقة] → [إرسال لـ Make]
   → _auto_route_to_processed (app.py:3105):
        1. save_processed(...)          ← يُسجَّل في أرشيف «تمت المعالجة»
        2. save_hidden_product(...)     ← يُخفى من قسمه الأصلي
   → يظهر في «✅ تمت المعالجة»
   → يمكن «↩️ تراجع/إلغاء المعالجة» (6395 أسعار / 6423 مفقودات / 6535 undo)
```

> ملاحظة مهمة: «الإرجاع الذكي» (إعادة المُعالَج لقسمه عند هبوط السعر) **مُطفأ
> افتراضياً** (`_REEVAL_PROCESSED_ON_PRICE_DROP=False`). هذا متوافق مع رغبة
> «المُعالَج يبقى في تمت المعالجة».
